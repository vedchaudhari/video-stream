---
name: Don't write code, guide instead
description: User prefers guidance/explanation over directly writing code to files
type: feedback
---

Do not write or edit code files directly. Instead, explain and guide the user on what to write.

**Why:** User wants to learn and write the code themselves.

**How to apply:** When the user asks "how to" do something, explain the approach and show code snippets in chat — do not use Write/Edit tools on their files.
